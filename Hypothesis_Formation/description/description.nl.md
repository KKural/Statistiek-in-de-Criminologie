# **Hypothese Formuleren**

Je bent een criminoloog die onderzoek doet naar de relatie tussen verschillende criminologische variabelen. Voor een nieuw onderzoek moet je een hypothese formuleren.

## **Scenario:**
Je hebt toegang tot een dataset met informatie over verdachten en hun misdrijven. Je wilt onderzoeken of er een verband bestaat tussen **leeftijd van verdachten** en **type misdrijf**.

## **Opdracht:**
Formuleer een hypothese die de mogelijke relatie tussen deze twee variabelen beschrijft. Je hypothese moet:

1. **Beide variabelen bevatten:** "leeftijd" en "misdrijf" (of synoniemen)
2. **Een richting aangeven:** bijvoorbeeld "jongere", "oudere", "meer", "minder"
3. **Het woord "associatie" of "verband" bevatten**

## **Voorbeeld van een goede hypothese:**
```
"Er is een associatie tussen leeftijd en type misdrijf, waarbij jongere verdachten meer geweldsdelicten plegen dan oudere verdachten."
```

### **Wat je moet doen:**
Schrijf je hypothese als een **string** en sla deze op in een variabele genaamd `hypothesis`.

```python
hypothesis = "Jouw hypothese hier..."
```

**Let op:** Je hypothese wordt gecontroleerd op de aanwezigheid van de belangrijkste termen en concepten!
