De **onderzoeksvraag** luidt: "Heeft de nieuwe interventie **effect** op het aantal meldingen van overlast?"

Wat voor soort **vraag** is dit?

1. Beschrijvende vraag (Wat?)
2. **Verschilvraag (Verschilt/Effect?)**
3. Samenhangvraag (Hangt samen?)
4. Voorspellingsvraag (Voorspelt?)

**Hint:** *Deze vraag vergelijkt **twee groepen** om een **effect** te meten.*

Typ je antwoord als één enkel getal (**1-4**) om je keuze aan te geven.
