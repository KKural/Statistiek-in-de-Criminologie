Welke uitspraak over Gamma is fout?

1. Gamma kan zowel positief als negatief zijn.
2. Gamma ligt tussen -1 en +1.
3. Gamma kan waarden boven 1 geven als veel observaties gelijk zijn.
4. Gamma negeert gebonden rangen (ties) bij de berekening.
