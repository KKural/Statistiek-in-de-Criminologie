# Dataset Verkenning: Veiligheidsgevoel in Gent

Je hebt toegang tot een dataset genaamd `df_fear_of_crime_gent` die gegevens bevat over het veiligheidsgevoel van inwoners in Gent.

## Opdracht

Gebruik de `str()` functie om de structuur van de dataset `df_fear_of_crime_gent` te bekijken.

## Instructies

- Typ de R-code om de structuur van de dataset te tonen
- De code die je moet schrijven is: `str(df_fear_of_crime_gent)`

## Wat je zult zien

De `str()` functie toont je:
- Het type van elke variabele (num, chr, Factor, etc.)
- Het aantal observaties (rijen)
- Het aantal variabelen (kolommen)
- Een voorbeeldweergave van de eerste waarden

## Context

Deze dataset bevat gegevens over:
- Demografische kenmerken van respondenten
- Veiligheidsgevoelens in verschillende situaties
- Ervaring met criminaliteit
- Percepties van de buurt
