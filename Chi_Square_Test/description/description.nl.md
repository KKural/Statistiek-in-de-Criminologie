# **Chi-kwadraat Test Uitvoeren**

Je hebt een dataset met gegevens over het geslacht van verdachten en het type misdrijf. De data is al voor je geladen in een matrix genaamd `data_matrix`.

## **De dataset ziet er zo uit:**
```
         Misdrijf
Geslacht  Diefstal Geweld Fraude
    Man       45     15     20
    Vrouw     25     35     40
```

## **Vraag:**
Voer een chi-kwadraat test uit om te onderzoeken of er een significant verband is tussen geslacht en type misdrijf.

**Opdracht:** Typ de juiste R-functie om een chi-kwadraat test uit te voeren op `data_matrix`.

## **Wat je moet typen:**
Gebruik de chi-kwadraat test functie van R.

## **Na het uitvoeren zie je het resultaat:**
Het systeem toont je de volledige uitvoer van de test, inclusief:
- Chi-kwadraat statistiek
- Degrees of freedom  
- P-waarde
- Interpretatie van het resultaat
