Welke uitspraak over Spearman's rho is fout?

1. Spearman's rho ligt altijd tussen -1 en +1.
2. Spearman's rho meet de monotone relatie tussen twee ordinale variabelen.
3. Spearman's rho kan negatief zijn als hogere waarden van de ene variabele samenhangen met lagere waarden van de andere.
4. Spearman's rho kan nooit negatief zijn; een daling in de ene variabele betekent automatisch een daling in de andere.
