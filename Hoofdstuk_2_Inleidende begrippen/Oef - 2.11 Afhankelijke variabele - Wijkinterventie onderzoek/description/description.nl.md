In hetzelfde **wijkinterventie onderzoek** wordt het **effect** van de interventie gemeten.

Wat is de **afhankelijke variabele** in dit onderzoek?

1. De type interventie
2. **Het aantal meldingen van overlast**
3. De wijk waar de jongeren wonen
4. Het aantal jongeren in het onderzoek

**Hint:** *De **afhankelijke variabele** is wat wordt gemeten als **uitkomst** van het onderzoek.*
