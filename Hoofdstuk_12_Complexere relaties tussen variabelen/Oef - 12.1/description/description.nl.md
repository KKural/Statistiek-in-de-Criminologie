In een padmodel onderscheiden we drie soorten variabelen: **exogene** variabelen (geen inkomende pijlen), **intermediaire** variabelen (ontvangen én sturen pijlen) en de **endogene** uitkomstvariabele (ontvangt pijlen, stuurt er geen).

Welk type variabele is **"Moreel oordeel (Stelen is OK)"** in dit padmodel?

1. Exogeen — geen inkomende pijlen vanuit andere modelvariabelen
2. Endogeen — de uiteindelijke uitkomstvariabele van het model
3. Intermediair — ontvangt én stuurt pijlen naar andere variabelen
4. Exogeen én intermediair — vervult beide rollen tegelijk
